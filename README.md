# Customer Segmentation using K-Means Clustering

## 📌 Overview
This project uses K-Means clustering to segment customers based on their income and spending behavior.

## 📊 Dataset
Mall Customers dataset

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🚀 Features
- Data preprocessing
- Elbow method for optimal clusters
- K-Means clustering
- Data visualization

## 📈 Results
Customers are grouped into 5 clusters:
- High income, high spending (VIP)
- Low income, low spending
- Target customers
